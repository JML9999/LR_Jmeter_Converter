Introduction¶

Loadrunner to Jmeter conversion is built in python2.7. The utility is capable of converting basic http/html methods in Loadrunner .c files to their equivalent Jmeter script.

How to use ?¶

  1. Run executable “LR_Parsing.exe” present in in dist/LR_Parsing
  2. Provide inputs as requested
    1. location of the LR file
    2. location to save the output
  3. Read through Warnings/errors listed on the console

Enhancing the generated jmeter script¶

What to expect ?¶

Converted jmeter script will have

  • predefined varaibles linked to the test plan
  • pre-populated parameters into http request defaults
  • CSV data set with parameter names for all the parameters reading data from file
  • A loop controller with iteration count set to 1
  • samples as per the test flow in Loadrunner
  • Parameterization, correlation, response assertions, lr_think_time with appropraiate conditions
  • If and while loops with true as a condition

Changes to do¶

  • Custom parameters types listed on the console are to be added manually
  • Parameters in web_custom_requests are to be handeled manually. i.e; replace {param1} to ${param1}
  • Any custom code or conditions in if/while
  • Listeners are to be added

Known issues¶

  • web_custom_requests with body extending to multiple lines are converted with less double quotes and more /
